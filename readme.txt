This is a plugin for filtering post results on administration control panel based on Authors.

== Description ==
This is a plugin for filtering post results on administration control panel based on Authors.It will display a drop down with the list of users having publish_post permission, so user can choose any from the list and view the result.

== Installation ==
1. Upload ”SIC Admin Author Filter” to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
